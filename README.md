# 2026 World Cup Prediction Simulation

- **Data source**: https://www.eloratings.net
- **Model**: Poisson regression & Monte Carlo Simulation
- **Input Data**:
  - `data/elo_ratings.json` (팀별 초기 Elo 레이팅)
  - `data/groups.json` (48개국 12개 조 편성표)
  - `data/actual_results.json` (실제 경기 스코어 기록)

## 프로젝트 개요 및 주요 기능
본 프로젝트는 Elo 레이팅과 푸아송 분포를 기반으로 2026년 월드컵(48개국 체제) 전체 과정을 10,000번 반복 시뮬레이션하여 각 국가의 토너먼트 진출 확률을 예측합니다.

### 1. 단일 경기 결과 예측 (`src/poisson.py`)
- 양 팀의 전력(Elo 레이팅) 차이를 바탕으로 각 팀의 예상 득점(Lambda)을 산출합니다.
- `numpy.random.poisson`을 사용하여 예상 득점을 실제 축구 스코어 형태(예: 2-1, 0-0)로 무작위 추출합니다.

### 2. 조별 리그 진행 (`src/simulation.py`)
- 조별(A~L조) 4개 팀이 서로 1번씩 맞붙는 풀리그(조당 6경기)를 진행합니다.
- 승점(승 3, 무 1, 패 0), 골득실차, 다득점을 계산하여 그룹별 1~4위 순위를 정렬합니다.

### 3. 32강 대진표 및 토너먼트 진행
- 각 조 1, 2위 24팀과 3위 중 상위 8팀(와일드카드)을 선별해 총 32팀을 구성합니다.
- 성적순 시드 배정으로 대진표를 만들고 단판 승부를 시뮬레이션합니다. 무승부 시 Elo 승률 기반의 확률적 승부차기로 반드시 승자를 결정합니다.

### 4. 실시간 데이터 자동 수집 및 혼합 반영 (`fetch_data.py`)
- eloratings.net에서 월드컵 참가국의 최신 실시간 ELO 레이팅과 완료된 실제 경기 결과를 동시에 수집하여 로컬 데이터(`elo_ratings.json`, `actual_results.json`)를 자동으로 최신화합니다.
- 실제 종료된 경기는 조별 리그 시뮬레이션 시 실제 점수로 고정(Override) 대입되어 더 정확한 진출 확률을 계산합니다.

### 5. 개별 국가 간 승부 예측기 (`predict_match.py`)
- 사용자가 선택한 두 국가의 현재 ELO 레이팅과 푸아송 분포를 기반으로 개별 맞대결의 승/무/패 확률을 산출하고, 넘파이(NumPy) 벡터 연산 기반의 1,000,000회 시뮬레이션을 통해 가장 발생 확률이 높은 대표 예상 스코어를 정밀하게 도출합니다.

### 6. 몬테카를로 시뮬레이션 (`main.py`)
- 앞서 구현한 "월드컵 전체 과정"을 10,000번 반복하여 각 국가의 진출 확률(%)을 통계로 추출합니다.

## 수치 산출 방식 및 공식 (Mathematics & Formulas)

본 시뮬레이션은 **Elo 레이팅**과 **푸아송 분포(Poisson Distribution)** 두 가지 수학적 모델을 결합하여 경기 결과를 예측합니다.

### 1. 기대 승률 산출 및 레이팅 업데이트 (Elo Rating System)
두 팀의 Elo 점수 차이를 바탕으로 각 팀의 승리 기대값(확률)을 산출합니다. (`src/elo.py`)

- **공식**:
  $$E_A = \frac{1}{1 + 10^{(R_B - R_A) / 400}}$$
  - $E_A$: 팀 A의 기대 승률 (0 ~ 1)
  - $R_A$: 팀 A의 Elo 점수, $R_B$: 팀 B의 Elo 점수
- **설명**: 두 팀의 점수차가 400점일 때, 점수가 높은 팀의 승률은 약 90.9%로 계산됩니다.
- **월드컵 가중치 (K-factor)**: 실제 경기 결과(`fetch_data.py`) 반영 시, 월드컵 본선 경기의 공식 중요도인 **`K-factor = 60`**을 대입하여 경기 결과가 각 팀의 전력(Elo)에 즉각적이고 뚜렷하게 반영되도록 업데이트합니다.

### 2. 팀별 예상 득점(Lambda, $\lambda$) 산출
Elo 기반 기대 승률을 축구 경기의 팀 평균 득점(Base Goals)을 활용해 '예상 득점 수치($\lambda$)'로 변환합니다. (`src/poisson.py`)

- **공식**:
  $$\lambda_A = \text{Base Goals} \times \left( \frac{E_A}{0.5} \right)$$
  - $\lambda_A$: 팀 A의 예상 득점 (푸아송 분포의 모수)
  - $\text{Base Goals}$: 한 팀의 평균 득점 기준값 (본 프로젝트에서는 **1.5골** 적용)
  - $E_A$: 팀 A의 기대 승률

### 3. 실제 스코어 확률 산출 (Poisson Distribution)
산출된 예상 득점($\lambda$)을 푸아송 확률 질량 함수에 대입하여, 한 팀이 특정 경기에서 $k$골을 넣을 확률을 계산합니다. 축구 경기의 득점은 독립적인 사건으로 가정합니다.

- **공식**:
  $$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}$$
  - $P(X = k)$: 한 팀이 정확히 $k$골을 득점할 확률
  - $\lambda$: 팀의 예상 득점, $e$: 자연상수 (약 2.718)
  - $k$: 득점 수 (0, 1, 2, 3...)
- **설명**: 시뮬레이션 단계에서는 위 수식을 확률 분포로 갖는 `numpy.random.poisson(lambda)` 함수를 사용하여 실제 스코어(예: 2-1, 0-0)를 난수로 추출해 경기를 진행합니다.

## 선수 부상 및 스쿼드 뎁스 보정 시스템 (Squad Depth & Injuries Adjustment)

본 시뮬레이션은 핵심 주전 선수의 부상으로 인한 전력 누수 및 이를 보완할 수 있는 각국의 **스쿼드 뎁스(선수층의 두터움)** 차이를 반영합니다.

### 1. 부상 선수 데이터 정의 (`data/injuries.json`)
선수의 포지션(공격/수비) 및 전력 기여도에 따라 등급을 설정하여 파일에 기록하면 시뮬레이션과 일대일 예측기에 자동 적용됩니다.
* **Tier S (대체 불가 에이스)**: 기본 전력 감쇄율 15% (0.15)
* **Tier A (주전 선발 멤버)**: 기본 전력 감쇄율 8% (0.08)
* **Tier B (주요 로테이션 멤버)**: 기본 전력 감쇄율 4% (0.04)

**예시 (`data/injuries.json`)**:
```json
{
  "South Korea": [
    {"name": "Son Heung-min", "position": "attack", "tier": "S"}
  ],
  "Spain": [
    {"name": "Rodri", "position": "defense", "tier": "S"}
  ]
}
```

### 2. 국가별 스쿼드 뎁스 계수 (Squad Depth Index)
에이스가 부상당하더라도 대체할 수 있는 백업 선수의 기량 차이에 따라 국가별로 계수(0.2 ~ 1.0)를 정의합니다.
* **0.2 (초강팀)**: 스페인, 프랑스, 잉글랜드, 독일 등 (에이스가 결장해도 백업이 월드클래스 수준)
* **0.8 (스타 의존팀)**: 대한민국, 호주, 캐나다 등 (에이스와 대체 백업과의 기량 격차가 매우 큼)
* **1.0 (일반/약팀)**: 그 외 백업 층이 얇아 에이스 결장 시 대체가 불가능한 국가 (기본값)

### 3. 최종 보정율 계산 방식
포지션에 따른 전력 누수율을 계산하여 양 팀의 예상 득점(람다, $\lambda$)에 교차 적용합니다:
* **팀 전력 누수율**: $\text{누수율} = \sum (\text{부상자 기본 감쇄율} \times \text{국가별 뎁스 계수})$
* **최종 공격/수비 보정 배율**:
  * 공격 배율 = $\max(0.5, 1.0 - \text{공격 누수율})$ (내 득점력을 낮춤)
  * 수비 배율 = $\min(2.0, 1.0 + \text{수비 누수율})$ (상대 득점력을 높임)
* **최종 예상 득점 ($\lambda$)**:
  * $\lambda_{A(\text{최종})} = \lambda_A \times (\text{A팀 공격 배율}) \times (\text{B팀 수비 배율})$
